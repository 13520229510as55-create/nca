## 🔍 常见错误类型

### 错误 1：Dockerfile 语法错误

**表现**：
```
ERROR: failed to solve: process "/bin/sh -c apt-get update" did not complete successfully
```

**解决方案**：
- 检查 Dockerfile 语法
- 确认所有命令正确
- 检查包名是否正确

### 错误 2：依赖安装失败

**表现**：
```
E: Unable to locate package fonts-noto-cjk
```

**解决方案**：
- 检查包名是否正确
- 确认 apt-get update 成功
- 检查网络连接

### 错误 3：权限问题

**表现**：
```
ERROR: (gcloud.builds.submit) PERMISSION_DENIED
```

**解决方案**：
- 启用 Cloud Build API
- 检查服务账户权限
- 确认项目权限

### 错误 4：资源不足

**表现**：
```
ERROR: Build timed out
```

**解决方案**：
- 增加构建超时时间
- 检查配额限制
- 优化 Dockerfile

### 错误 5：内存不足

**表现**：
```
ERROR: failed to solve: executor failed running: out of memory
```

**解决方案**：
- 增加构建机器内存
- 优化构建步骤
- 分阶段构建
