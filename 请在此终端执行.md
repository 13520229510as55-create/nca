## 若使用服务账号部署（替代 gcloud auth login）

若希望完全用服务账号执行部署，需要先为该服务账号授予以下角色：

1. 打开 [GCP IAM 控制台](https://console.cloud.google.com/iam-admin/iam?project=n8n-video-490705)
2. 找到 `n8n-video@n8n-video-490705.iam.gserviceaccount.com`
3. 编辑 → 添加角色：
   - **Cloud Build 编辑者**
   - **Cloud Run 管理员**
   - **存储管理员**
   - **Service Usage 管理员**（用于启用 API）

然后修改 `deploy-now.sh`，在「检查认证状态」之前添加：
```bash
$GCLOUD auth activate-service-account --key-file="$SA_KEY_FILE"
```
